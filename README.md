# week10-day3-basic-pages

small multi-page React app built while learning routing with `react-router-dom`. Contains simple `Home`, `About`, and `Contact` pages and a shared `Navbar` + `Footer`.

How to run:

```bash
npm install
npm run dev
# build for production
npm run build
```

Notes: Contact page is UI-only (no backend). Student-level project focusing on routing and layout.
